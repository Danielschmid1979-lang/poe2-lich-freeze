# Lich Freeze + Herald Last Lament Build (PoE2)

## Class / Ascendancy
- Class: Acolyte
- Ascendancy: Lich

## Passive Tree Highlights
- Lich core nodes: ES, Low Life, Mana-as-Life
- Cold / Freeze clusters
- Hypothermia
- Herald scaling (Elemental damage, AoE)
- Mana efficiency / Reduced Mana cost
- Pathing: Start at Lich base → Cold/Freeze → Mana efficiency → Herald → ES / survivability nodes

## Skills
- **Main Attack: Last Lament Crossbow**
  - Frost Support (guaranteed Freeze)
  - Added Cold
  - Hypothermia
  - Increased Elemental Damage
  - Mana Efficiency Support
  - Reduced Mana Cost Support
- **Heralds**
  - Herald of Ice (Freeze Shards)
  - Herald of Thunder (Chain AoE)
  - Herald of Ash (Chaos/Burn)
  - Optional: Herald of Fire
- **Utility**
  - Discipline (ES buffer)
  - Frostbite / Chill Aura
  - Molten Shell / Aegis Shield
  - Mind over Matter (optional hybrid)

## Gear
- Weapon: Last Lament Crossbow (+Cold, Crit, Mana Efficiency)
- Offhand: Threaded Light / Scepter
- Helmet: +ES, +Mana Regen, +Cold Damage
- Chest: +ES, +Mana Regen
- Gloves: +Attack/Cast Speed, +Cold Damage
- Boots: +Move Speed, +Mana Regen
- Amulet: Beacon of Azis
- Rings: Thief's Torment (Mana on Hit)
- Belt: +ES, +Mana Regen
- Flasks: Lavianga’s Spirits (Infinite Cast), Defensive Flask (Instant ES)

- Infinite Cast
